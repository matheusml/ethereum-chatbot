Ethereum Chatbot
=====

### Back-end
Enter the folder:
```
cd back-end
```

Install the dependencies
```
npm i
cd functions
npm i
```

To run the tests, go the `back-end` folder and run:
```
npm t
```

To run the app, type:
```
firebase serve --only functions,hosting
```

The aplication is hosted at: https://ethereum-chatbot.firebaseapp.com/messages
